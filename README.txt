== Description ==
Using WP GDPR Ultimate plugin you can easily make your Wordpress website GDPR Compliance.

First Automated GDPR plugin for WordPress that supports most WordPress plugins and forms such as WooCommerce , Gravity Forms and more ...

Wordpress GDPR Ultimate helps you as Site or Business owner can easily inform your website visitors and customers about your business GDPR policy! and build trust with customers.

- Including GDPR-Checkbox in forms: According to GDPR guidelines, you as business/site owner have to inform visitors/customers about obtaining their data via ANY FORM and must get acceptance from visitors/customers. And therefore i have included GDPR-Checkbox function in this plugin and it works automatically using JS.

- Responsive GDPR Notification bar: Using this plugin you can easily show a notification bar in front-end and inform website Visitors/Customers about GDPR compliance and policy.

- "New" Add GDPR & Privacy Policies page link: You can easily select and include yours GDPR & Privacy Policies page link to Checkbox and GDPR Notification from plugin options page.
please check [WP GDPR ULTIMATE Website](https://aliayubi.com/wp-gdpr-ultimate/ "Plugin Website")

== Installation ==
1. Upload `gdpr_ultimate.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Find WP GDPR Ultimate in Admin Panel and Configure it.

== Frequently Asked Questions ==
= I seeing GDPR checkbox in a unnecessary form, how can i remove it? =
Firstly Find out class or ID for the form that you want exclude and then add those id or classes to WP-ADMIN->WP GDPR Ultimate->Exclude form by ID or Class field
